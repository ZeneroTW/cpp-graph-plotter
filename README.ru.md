# 📈 Построитель графиков функций

> 🇬🇧 [Read in English](README.md)

Десктопное приложение для построения графиков математических функций. Пользователь вводит функцию от `x`, задаёт диапазон, и программа отрисовывает интерактивный график в окне с осями координат.

Написано на **C++** с использованием **SFML** (графика) и **muParser** (парсинг математических выражений).

---

## Возможности

- Ввод произвольной математической функции от `x` во время запуска (например `sin(x)`, `x^2`, `x*cos(x)`)
- Задание пользовательского диапазона по оси X
- Автомасштабирование: диапазон Y вычисляется автоматически по значениям функции
- Оси координат со стрелками отображаются если попадают в видимую область
- Подпись функции в левом верхнем углу окна
- Корректная обработка неопределённых значений (NaN, бесконечность, деление на ноль)

---

## Зависимости

| Библиотека | Назначение |
|---|---|
| [SFML 2.x](https://www.sfml-dev.org/) | Окно, рендеринг, графика |
| [muParser](https://beltoforion.de/en/muparser/) | Парсер математических выражений |

Установка на Ubuntu:
```bash
sudo apt install libsfml-dev libmuparser-dev
```

---

## Сборка и запуск

```bash
# Компиляция
g++ main.cpp -o graph -lsfml-graphics -lsfml-window -lsfml-system -lmuparser

# Запуск (arial.ttf должен быть в той же папке)
./graph
```

Затем следуй подсказкам в терминале:
```
Введите функцию от x: sin(x)
Введите минимум X: -6.28
Введите максимум X: 6.28
```

## Поддерживаемый синтаксис

muParser поддерживает стандартные математические выражения:

| Пример | Описание |
|---|---|
| `sin(x)`, `cos(x)`, `tan(x)` | Тригонометрические функции |
| `x^2`, `x^3` | Степени |
| `sqrt(x)` | Квадратный корень |
| `exp(x)`, `log(x)` | Экспонента и логарифм |
| `x*sin(x) + 2` | Составные выражения |
